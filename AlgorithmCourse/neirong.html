课程内容

课程教学目标

1.通过对常用的、有代表性的算法的研究，让学生理解并掌握算法设计的基本技术。
2.培养学生分析算法复杂度的初步能力，锻炼其逻辑思维能力和想象力，并使之了解算法理论的发展。
3.鼓励学生运用算法知识解决各自学科的实际问题，培养他们的独立科研的能力和理论联系实践的能力。

基本教学内容

第一讲、算法分析与设计基础
  1. 教学目的与要求：介绍课程和背景知识
  2. 教学重点：课程简介
  3. 教学难点：算法复杂性背景知识
  4. 教学内容：课程简介，主讲内容概述、算法复杂性概述
  5. 课件：课件1
第二讲、分治策略
  1. 教学目的与要求：分治策略基本思想
  2. 教学重点：分治策略
  3. 教学难点：二维峰值查找算法
  4. 教学内容：基于峰值查找算法介绍分治策略
  5. 课件：课件2
第三讲、二叉搜索树
  1. 教学目的与要求：二叉搜索树
  2. 教学重点：二叉搜索树
  3. 教学难点：平衡的二叉搜索树——AVL树
  4. 教学内容：基于二叉搜索树的跑道预订求解问题
  5. 课件：课件3
第四讲、哈希策略
  1. 教学目的与要求：哈希策略
  2. 教学重点：哈希策略
  3. 教学难点：哈希冲突解决
  4. 教学内容：基于字符串之间的匹配问题介绍哈希策略
  5. 课件：课件4
第五讲、排序
  1. 教学目的与要求：排序与大师定理
  2. 教学重点：排序算法
  3. 教学难点：堆排序、快速排序、大师定理
  4. 教学内容：插入排序、堆排序、快速排序、基数排序
  5. 课件：课件5
第六讲、图
  1. 教学目的与要求：图
  2. 教学重点：图的基本概念
  3. 教学难点：图定义
  4. 教学内容：图论基础，包括图定义、应用等
  5. 课件：课件6
第七讲、图搜索
  1. 教学目的与要求：图搜索
  2. 教学重点：宽度优先搜索和深度优先搜索
  3. 教学难点：宽度优先搜索和深度优先搜索
  4. 教学内容：宽度优先搜索和深度优先搜索
  5. 课件：课件7
第八讲、最短路径
  1. 教学目的与要求：最短路径
  2. 教学重点：最短路径算法
  3. 教学难点：Dijkstra算法、Bellman-Ford算法、Floyd-Warshall算法、Johnson算法
  4. 教学内容：经典最短路径算法
  5. 课件：课件8
第九讲、贪心算法
  1. 教学目的与要求：贪心算法
  2. 教学重点：贪心策略
  3. 教学难点：最小生成树算法
  4. 教学内容：经典贪心算法实例
  5. 课件：课件9
第十讲、动态规划策略
  1. 教学目的与要求：动态规划策略
  2. 教学重点：动态规划策略
  3. 教学难点：背包问题
  4. 教学内容：经典动态规划算法实例
  5. 课件：课件10
第十一讲、NP完全
  1. 教学目的与要求：NP完全
  2. 教学重点：NP完全理论
  3. 教学难点：常见NP完全问题
  4. 教学内容：NP完全理论、图灵机模型
  5. 课件：课件11
