课程介绍

课程概述

计算机算法作为计算机解决实际问题的主要手段，是软件工程专业学生需要掌握的核心技能之一。算法分析与设计课程在数据结构课程的基础上，进一步剖析各类主要算法的问题描述、主要思想、基本原理和应用场景，使得学生不仅能掌握当前主要算法技术的工作原理，而且能理解该算法产生的背景和设计理念，使其能灵活运用这些技术，结合实际情况设计合适的算法解决问题。

课程内容

基础篇

第一讲、知识图谱的基本概念
  1. 教学目的与要求：介绍知识图谱的基本概念
  2. 教学重点：课程简介
  3. 教学难点：知识图谱的基本概念
  4. 教学内容：主要介绍知识图谱的基本概念、历史沿革、研究意义、应用价值等。
  5. 课件：课件1。

第二讲、知识图谱基础知识
  1. 教学目的与要求：主要介绍知识图谱密切相关的基础知识
  2. 教学重点：知识表示的方法、机器学习算法、自然语言处理
  3. 教学难点：机器学习算法基础
  4. 教学内容：主要介绍知识图谱密切相关的基础知识，包括知识表示的方法、机器学习算法、自然语言处理的相关概念和算法等方面的内容。
  5. 课件：课件2。

构建篇

第三讲、概念图谱构建
  1. 教学目的与要求：介绍一类重要的知识图谱——概念图谱
  2. 教学重点：isA关系的抽取、补全以及纠错
  3. 教学难点：isA关系补全及纠错
  4. 教学内容：包括常见的概念图谱介绍、isA关系抽取的方法、isA关系补全、isA关系纠错等方面的内容。
  5. 课件：课件3。

第四讲、百科图谱构建
  1. 教学目的与要求：介绍一类重要的知识图谱——百科图谱
  2. 教学重点：百科图谱的构建与融合
  3. 教学难点：百科图谱的构建
  4. 教学内容：包括百科图谱的定义和分类、基于单源的百科图谱构建方法、基于多源的百科图谱融合方法等方面的内容。
  5. 课件：课件4。

第五讲、知识图谱的众包构建
  1. 教学目的与要求：介绍众包技术在知识图谱构建中的应用
  2. 教学重点：基于众包的知识图谱构建与精化
  3. 教学难点：基于众包的知识图谱精化
  4. 教学内容：包括众包的概念、众包研究的问题、基于众包的知识图谱构建与精化等方面的内容
  5. 课件：课件5。

管理篇

第六讲、知识图谱数据管理
  1. 教学目的与要求：常见知识图谱数据数据模型、查询模型以及管理系统
  2. 教学重点：RDF图数据模型与SPARQL查询语言，属性图数据模型与Cypher查询语言
  3. 教学难点：RDF图数据管理系统gStore、属性图数据管理系统Neo4j
  4. 教学内容：gStore、RDF图数据模型与SPARQL查询语言；Neo4j、属性图数据模型与Cypher查询语言；JanusGraph、TinkerPop图计算框架与Gremlin图遍历语言及遍历机。
  5. 课件：课件6。

第七讲、大数据时代的知识图谱数据管理
  1. 教学目的与要求：常见分布式知识图谱数据管理系统
  2. 教学重点：集群式分布式知识图谱数据管理系统、联邦型分布式知识图谱数据管理系统
  3. 教学难点：gStoreD
  4. 教学内容：集群式分布式数据库系统：包括基于云平台和基于数据划分的；联邦型分布式知识图谱数据管理系统。
  5. 课件：课件7。
